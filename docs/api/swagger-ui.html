NGX Voice Sales Agent API

Documentación interactiva de la API del agente de ventas conversacional

v1.0.0

🚀 Inicio Rápido

Para comenzar a usar la API:

  • Obtén tus credenciales de acceso contactando al equipo de NGX
  • Autentica usando el endpoint /auth/login para obtener tu JWT token
  • Incluye el token en el header Authorization: Bearer {token}
  • Explora los endpoints disponibles abajo

📊 Características Principales

  • Conversaciones en tiempo real: Gestión completa de conversaciones de venta con IA
  • ML Predictivo: Predicción de objeciones, necesidades y probabilidad de conversión
  • Optimización inteligente: Motor de decisiones con <20ms de latencia
  • Analytics avanzado: Métricas detalladas y análisis de rendimiento

🔐 Autenticación

Todos los endpoints requieren autenticación JWT, excepto:

  • POST /auth/login - Iniciar sesión
  • POST /auth/register - Registrar nuevo usuario
  • GET /metrics - Métricas Prometheus (solo desarrollo)
